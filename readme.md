# AtCoder

## Environment

- C++20 (GCC 12)

## Usage

### Create a new answer

```console
c=abc000 q=a make new
```

### Create a new test input

```console
c=abc000 q=a make new_test
```

(with Pasteboard content)

```console
pbpaste | c=abc000 q=a make new_test
```

### Run with all test inputs

```console
c=abc000 q=a make test
```
